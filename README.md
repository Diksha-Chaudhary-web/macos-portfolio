# macOS Portfolio

A stylish macOS-inspired portfolio website built with React and Vite.

## Features
- Interactive macOS-style UI
- Project showcase
- Resume viewer
- Terminal CLI
- Spotify integration

## Tech Stack
React • Vite • Sass • JavaScript

## Setup
npm install
npm run dev

## Deploy
npm run build

## 🚀 Installation & Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
